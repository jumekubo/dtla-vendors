Settings — GitHub connection

To save edits back to the repo (and trigger the auto-update workflow), paste a GitHub fine-grained personal access token scoped to this repository with Contents: Read & write and Actions: Read & write permissions. Create one at github.com/settings/personal-access-tokens/new. The token is stored only in this browser (localStorage), never in the repo.

Update vendor info from the web

Option A — Automatic (recommended once set up). Triggers a GitHub Action that researches every vendor online using Google Gemini's free API (checks phone numbers, addresses, hours, brands carried, whether they've closed) and opens a pull request with proposed changes for you to review and merge. Requires the GEMINI_API_KEY repo secret and a token in Settings.

Option B — Via Claude. Copies a ready-made research prompt (including your current vendor data) to the clipboard. Paste it into a Claude / Cowork session and Claude will research and hand back an updated vendors.json.
